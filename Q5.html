Html vs Html5

Html Html5
1.Older version of the markup language used to structure web content. 1.Latest version with new features for modern web development.
2.No support for native audio and video embedding. 2.Supports native <audio> and <video> tags to embed media without plugins.
3.Relies heavily on non-semantic elements like <div> and <span>. 3.Introduces semantic elements like <header>, <footer>, <article>, and <section>.
4.Depends on external plugins (e.g., Flash) for rich media or animations. 4.Includes built-in APIs like Canvas, Web Storage, and Geolocation for multimedia and interactivity.
5.Limited support for mobile and responsive design. 5.Designed with mobile-first and responsive principles in mind.
